# Color
## Accessibility
There are two primary accessibility guidelines related to color:

1. Always use a text color that provides sufficient contrast (4.5:1 or greater[^1])
with its background. Pragramtically, this means never using a gray lighter than
Sonic Silver (`color(gray)`) against a white background
2. Never use color along to indicate a condition or state[^2]â€”consider using an icon
or more descriptive text in such cases
