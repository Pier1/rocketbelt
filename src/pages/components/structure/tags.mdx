# Tags
Use tags to augment a larger piece of the design with a smaller bit of data.

## Basic Variations
```html
<aside class="tag">Indoor/Outdoor</aside>
<aside class="tag">Online Only</aside>
<aside class="tag tag-sale-clearance">Sale</aside>
<aside class="tag tag-sale-clearance">Clearance</aside>
<aside class="tag tag-new">New</aside>
```

## Promo Tags
```html
<div class="tag tag-promo">Pier 1 Picks: 25% Off</div>
```
