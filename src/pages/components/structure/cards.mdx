# Cards
Cards provide rudimentary visual structure to content. Their composition and
contents can vary greatly, and they can be quickly arranged on a <Link to="/components/structure/flexible-grid">Flexible Grid</Link>.

## Basic Cards

```html
<div class="grid-fluid">
  <div class="row xs-center">
    <div class="col xs-3">
      <div class="card">
        Cards are building blocks that lead to more detailed information. Their
        use, size, and composition can vary greatly.
      </div>
    </div>
    <div class="col xs-3">
      <div class="card">
        Cards may be organized using flex grid classes, and can contain many
        types of content.
      </div>
    </div>
  </div>
</div>
```

## Action Cards
Use an Action Card to give a strong visual association between Card content and
a <Link to="/components/actions-and-feedback/links#ct-as">CTA</Link> or desired
action.

```html
<div class="card card-action z-depth-2">
  <div class="card-content">
    <div class="card-title">Action Cards</div>
    <div class="card-body">
      More structured, with a title and one or more actions a user can take.
    </div>
  </div>
  <div class="card-actions">
    <button class="button button-text">Button 1</button>
    <button class="button button-text">Button 2</button>
  </div>
</div>
```

## Depth
Cards can benefit from the extra definition and visual interest that some depth
provides. Add a `z-depth` class or include the `z-depth` mixin in your SCSS.

### Class-Based
Add any of the classes below to a card.
<ExampleCard className="card z-depth-0">z-depth-0</ExampleCard>
<ExampleCard className="card z-depth-1">z-depth-1</ExampleCard>
<ExampleCard className="card z-depth-2">z-depth-2</ExampleCard>
<ExampleCard className="card z-depth-3">z-depth-3</ExampleCard>

### Mixin-Based
The `z-depth` mixin is also provided via Rocketbelt Tools. Use this mixin if
you need to override any of the properties and you'd prefer to keep the DOM a
little cleaner.

```scss code-only
// Use an integer between 0 and 4 for increasing depth
@include z-depth(2);
```

## Full Bleed Cards
Full Bleed Cards allow an image to take center stage, optionally augmented with
a caption.

```html
<div class="card card-full-bleed z-depth-1">
  <div class="">
    <img class="" src="https://images.pier1.com/dis/dw/image/v2/AAID_PRD/on/demandware.static/-/Sites-pier1_master/default/images/PS73156/PS73156_1.jpg?sw=400&amp;sh=400" />
  </div>
  <div class="card-content">
    <div class="card-title">Full Bleed Cards</div>
    <div class="card-body">
      <p>Combine an image with card content.</p>
    </div>
  </div>
</div>
```

## Thumbnail Cards
Use Thumbnail Cards to provide a representative image for a Card, or to promote
a strong association between an image and the Card content.

### Constrained via Grid
```html
<div class="card card-image card-image-thumbnail z-depth-1">
  <div class="card-image_image">
    <img class="" src="https://www.pier1.com/dis/dw/image/v2/AAID_PRD/on/demandware.static/-/Sites-pier1_master/default/dwb96c5cfa/images/4036802/4036802_1.jpg?sw=225&sh=225" />
  </div>
  <div class="card-content">
    <div class="card-title">Thumbnail Cards</div>
    <div class="card-body">
      Put content and an image side-by-side. Constrain the card's width
      via the grid…
    </div>
  </div>
</div>
```

### Constrained via CSS
```html
<div class="card card-image card-image-thumbnail z-depth-1" style="max-width: 33%;">
  <div class="card-image_image">
    <img style="width: 80px;" src="https://www.pier1.com/dis/dw/image/v2/AAID_PRD/on/demandware.static/-/Sites-pier1_master/default/dwb96c5cfa/images/4036802/4036802_1.jpg?sw=225&sh=225" />
  </div>
  <div class="card-content">
    <div class="card-title">Thumbnail Cards</div>
    <div class="card-body">… or via widths on the card itself.</div>
  </div>
</div>
```
