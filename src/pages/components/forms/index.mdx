# Forms

## Tips
- Form inputs should always be paired with a label element—but the label element
  may be `visually-hidden` if the purpose of the input is clear
- Don't use the placeholder attribute as a substitute for a label. It's a misuse
  of the attribute and leaves you with an inaccessible, unlabeled field.
- Always use a label's `for` attribute to link it to its input. This guarantees
  a more touch-friendly interface as tapping the label will bring focus to its
  input.

## Accessibility
- a11y/JS around validation messages
- Never use placeholder text instead of a label. Placeholders alone render an
  input that is inaccessible and whose purpose is unclear when the input is
  focused

## Standard Form Elements
```html
<form>
  <div class="form-group">
    <input id="basic-fname" type="text" />
    <label for="basic-fname">First Name</label>
  </div>
  <div class="form-group">
    <input id="basic-email" type="email" />
    <label for="basic-email">Email Address</label>
  </div>
  <div class="form-group">
    <input id="basic-tel" type="tel" />
    <label for="basic-tel">Phone Number</label>
  </div>
  <div class="form-group">
    <input id="basic-password" type="password" />
    <label for="basic-password">Password</label>
  </div>
  <div class="form-group">
    <textarea id="basic-message"></textarea>
    <label for="basic-message">Message</label>
  </div>
  <div class="form-group">
    <select id="basic-select">
      <option value="" disabled selected hidden>
        Select a collection
      </option>
      <option>Biblioteca</option>
      <option>Marcel</option>
      <option>Pierce</option>
    </select>
    <label for="basic-select">Collection</label>
  </div>
</form>
```

## Invalid Elements
```html
<form>
  <div class="form-group">
    <input id="invalid-email" type="email" value="not|a=valid!email" />
    <label for="invalid-email">Email Address</label>
    <small class="validation-message">Please enter a valid email address.</small>
  </div>
  <div class="form-group">
    <input class="invalid" id="invalid-phone" type="tel" value="(not) a-phone-number" />
    <label for="invalid-phone">Phone Number</label>
    <small class="validation-message">Please enter a valid phone number.</small>
  </div>
  <div class="form-group">
    <input class="invalid" id="invalid-password" type="password" value="doesn't-meet-regex" />
    <label for="invalid-password">Password</label>
    <small class="validation-message">Please enter a valid password.</small>
  </div>
  <div class="form-group">
    <input class="invalid" id="invalid-fname" type="text" />
    <label for="invalid-fname">First Name</label>
    <small class="validation-message">Please enter a first name.</small>
  </div>
  <div class="form-group">
    <textarea class="invalid" id="invalid-message"></textarea>
    <label for="invalid-message">Message</label>
    <small class="validation-message">Please enter a message.</small>
  </div>
  <div class="form-group">
    <select class="invalid" id="invalid-select">
      <option value="" disabled selected hidden>Select a collection</option>
      <option>Biblioteca</option>
      <option>Marcel</option>
      <option>Pierce</option>
    </select>
    <label for="invalid-select">Collection</label>
    <small class="validation-message">Please select a collection.</small>
  </div>
</form>
```

## Helper Text
```html
<form>
  <div class="form-group">
    <input id="helper-password" type="password" />
    <label for="helper-password">Password</label>
    <div class="messages"><small class="helper-text">At least 8 characters</small></div>
  </div>
  <div class="form-group">
    <input class="invalid" id="helper-fname" type="tel" />
    <label for="helper-fname">Phone number</label>
    <div class="messages"><small class="validation-message">Please enter a valid Phone Number</small><small class="helper-text">US numbers only, please</small></div>
  </div>
</form>
```

## Required vs. Optional Fields
```html
<form>
  <div class="form-group">
    <input id="required-email" type="email" required value="help@pier1.com" />
    <label for="required-email">Email Address</label><small class="helper-text">Please enter a valid email address.</small>
  </div>
  <div class="form-group">
    <input id="required-mobile" value="(555) 555-1212" />
    <label for="required-mobile">Mobile Phone</label><small class="helper-text">Please enter a valid Mobile Phone.</small>
  </div>
</form>
```
