# Messages, Notices & Alerts

## Messages
Messages indicate the state of the page or of a large UI component when the
page loads.
```html
<aside class="message message-info">
  <svg class="message_icon icon" role="img">
    <use xlink:href="/icons/rocketbelt.icons.svg#rb-icon-info"></use>
  </svg>
  <section class="message_body">
    <span class="message_title">
      This is the message title.
    </span>
    <span class="message_text">
      This is the message text.
    </span>
  </section>
</aside>
```

## Notices
Notices indicate the status of a UI component *after an action is taken* on
the page (and without a page reload). They also indicate the results of actions
that can be undone, tried again, or corrected.

Use a Notice when form validation alone may not be the most appropriate way to
acknowledge the UI's state after user input.

Notices come in three variations:
**Default**, **Attached**, and **Applied**.

### Default
```html
<figure class="notice">
  <div class="notice_text text-with-icon">
    The promotion wasn't recognized.
    <svg class="icon icon-left" aria-hidden>
      <use
        xlink:href="/icons/rocketbelt.icons.svg#rb-icon-error"
      ></use>
    </svg>
  </div>
  <button class="button button-minimal notice_close">
    <svg class="icon">
      <use
        xlink:href="/icons/rocketbelt.icons.svg#rb-icon-close"
      ></use>
    </svg>
  </button>
</figure>
```
### Attached
Use an Attached Notice to acknowledge that the user's action may not yet be
validâ€”but that it was [understood]{.emphasize} and will be [remembered]{.emphasize}.
```html
<figure class="notice notice-attached">
  <div class="notice_text text-with-icon">
    You must use a specific payment method to qualify for this promotion.
    <svg class="icon icon-left" aria-hidden>
      <use
        xlink:href="/icons/rocketbelt.icons.svg#rb-icon-warning"
      ></use>
    </svg>
  </div>
  <button class="button button-minimal notice_close">
    <svg class="icon">
      <use
        xlink:href="/icons/rocketbelt.icons.svg#rb-icon-close"
      ></use>
    </svg>
  </button>
</figure>
```

### Applied
Use an Applied Notice to indicate that one or more of the Notice's prior states
have been [resolved]{.emphasize} or altered.
```html
<figure class="notice notice-applied">
  <div class="notice_text text-with-icon">The promotion was removed.</div>
  <button class="button button-text notice_undo">Undo</button>
</figure>
```
