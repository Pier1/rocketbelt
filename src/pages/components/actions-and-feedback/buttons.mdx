# Buttons & Links

## Default/Secondary Buttons
Anchors go places, buttons do things

```html
<button class="button">Secondary</button>
<button class="button" disabled>Disabled</button>
<button class="button button-sm">Small</button>
<button class="button button-mini">Mini</button>
```

## Primary Buttons
* Convey or a "Desired" action
* One per page/major page component
* When Primary & Secondary buttons are grouped together, always place the Primary button on the right

```html
<button class="button button-primary">Primary</button>
<button class="button button-primary" disabled>Disabled</button>
<button class="button button-sm button-primary">Small</button>
<button class="button button-mini button-primary">Mini</button>
```

## Text/Minimal Buttons
* Can represent a primary action if placed by itself in a visually small component
* If grouped with another button, always represents a secondary action

```html
<button class="button button-lg button-text">Text Button</button>
<button class="button button-lg button-text" disabled>Disabled</button>
<button class="button button-md button-text">Medium Text</button>
<button class="button button-sm button-text">Small Text</button>
<button class="button button-mini button-text">Mini Text</button>
```

## Anchor Buttons
* Why `role=button` is important
* Use these sparingly
* Anchors go places, buttons do things

```html
<a class="button button-lg button-primary" href="#" role="button">
  Anchor Button
</a>
<a class="button button-lg button-primary" disabled href="#" role="button">
  Disabled
</a>
<a class="button" href="#" role="button">
  Medium Anchor
</a>
<a class="button button-sm button-primary" href="#" role="button">
  Small Anchor
</a>
<a class="button button-mini button-primary" href="#" role="button">
  Mini Anchor
</a>
```

## Block Buttons
* Mobile usage
* Use the grid to auto-size them and allow them to wrap

```html
<div class="grid-fluid">
  <div class="row">
    <div class="col xs-12 md-6">
      <button class="button button-block">Block</button>
    </div>
    <div class="col xs-12 md-6">
      <button class="button button-primary button-block">Block</button>
    </div>
  </div>
</div>
```
