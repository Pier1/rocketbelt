# Dropdowns

## Accessibility
Include the file `rocketbelt.dropdowns.js` and dropdowns will be augmented with
accessibility metadata when the page loads. If a dropdown is added to the page
after it loads, calling `rb.dropdowns.decorateDropdowns()` will add the
appropriate accessibility attributes. Dropdowns can also be navigated with the
keyboard.

## Basic
```html astronauts
<div class="dropdown">
  <span class="dropdown_label">Select an Apollo astronaut:</span>
  <div class="dropdown_body">
    <button class="dropdown_button"></button>
    <ul class="dropdown_list">
      <li class="dropdown_option">Neil Armstrong</li>
      <li class="dropdown_option">Buzz Aldrin</li>
      <li class="dropdown_option">Alan Shepard</li>
      <li class="dropdown_option">Jim Lovell</li>
      <li class="dropdown_option">Michael Collins</li>
      <li class="dropdown_option">Gus Grissom</li>
    </ul>
  </div>
</div>
```

## Content Overflow
Keep longer or taller lists from getting out of hand by adding a `max-height`
property to a selector that targets `.dropdown_list`.

```scss code-only
// Your selector may need to be more specific than this!
.dropdown_list {
  max-height: 360px;
}
```

## JavaScript
<h3 class="code">decorateDropdowns</h3>

`rb.dropdowns.decorateDropdowns()` can be called manually to add accessibility
properties to a Dropdown. This function runs on page load for all Dropdowns that
are present.

<h3 class="code">toggleDropdown</h3>

Use `rb.dropdowns.toggleDropdown($dropdown)` to programmatically open or close a
dropdown. Be sure the `$dropdown` parameter passed in is a jQuery object
constructed from a `.dropdown` top-level class.

<h3 class="code">focusItem</h3>

`rb.dropdowns.focusItem` sets the value of the list of options. It requires a
`$dropdownOption` parameter—be sure this is a jQuery object constructed from a `.dropdown_option`—and takes an optional configuration object:

```js code-only
{
  // Both options default to false if omitted.
  // Closes an open Dropdown after the function runs
  closeAfterFocus: yourCloseValue,
  // "Selects" the option by copying the option's HTML into .dropdown_button
  selectAfterFocus: yourSelectValue,
}
```
