Color

Rocketbelt includes a color palette with generated secondary & accent colors. Color can be used to attract attention to specific pieces of the design, or to indicate the state of an element on the screen.

UX Tip
Use color to guide the eye to a preferred action or to the most important information.

The tools SASS partial provided by Rocketbelt includes a color helper function to make color usage foolproof. Not only will you never need to memorize another hex code, if a functional color definition changes all usages of the color function will update automatically. This means much lower friction & fewer repeated color definitions!

Toolbelt
Always use the color(…) function to declare color values.

Even users without a vision impairment can have a hard time with low-contrast elements. Be sure that text has sufficient contrast with its background. WCAG 2.0 Level AA dictates a ratio of at least 4.5:1. Use tools like the Tanaguru Contrast Finder to check a layout's contrast.

Be an  a11y
Never use color as the sole indicator of information.
