Typography

Rocketbelt's typography is built on a modular scale. The scale makes it easy to use type to visually reinforce the hierarchy of content. Rocketbelt also includes plenty of helper classes to support icons, call-to-action links, lists, and pull quotes.

Headings

Headings are responsive and use a modular scale—this means that a giant heading on desktop isn't quite so dominant on a smaller viewport. See A More Modern Scale for Web Typography for background information on this design decision.

h1. Pier 1 Imports

h2. Pier 1 Imports
