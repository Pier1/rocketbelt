Tables

Tables aren't suitable for layout on the modern web but they still serve an important purpose.

Be an  a11y
When building a table…
  • Always use a <caption> or an aria-describedby attribute to make the table's purpose clear.
  • Always identify column & row headers with the <th scope="…"> element.
  • Don't use absolute (pixel) sizing for table content.
