Alerts

Flash Messages

Congratulations! This is a success message.
Caution! This is a warning message.
Look out! This is an error message.
Just FYI. This is an info message.

                              <div class="message-flash message-success">Congratulations! This is a success message.</div>
                              <div class="message-flash message-warning">Caution! This is a warning message.</div>
                              <div class="message-flash message-error">Look out! This is an error message.</div>
                              <div class="message-flash message-info">Just FYI. This is an info message.</div>

Dismissable Flash Message

The action completed successfully. 

                              <div class="message-flash message-success message-dismissable">The action completed successfully.
                                <button class="message-dismissable_close"></button>
                              </div>

                              <script src="./rocketbelt.alerts-dismissable.min.js"></script>
