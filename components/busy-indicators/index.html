Busy Indicators

Component-Level

"Is Busyable"

Whenever possible, add the is-busyable class to the container that wraps components that may be marked as busy. This will maximize performance by greatly narrowing the scope of DOM elements that will be watched by a MutationObserver. If no element matches .is-busyable, the observer will be registered against body.

Custom Sizing

If a large component needs to indicate a busy state, supply a custom rule that targets the font-size of .is-busy_overlay. (This element is created & managed via Javascript.) Be sure to use Rocketbelt's font-size() function!

For example:


                              #component-wrapper .is-busy_overlay {
                                font-size: font-size(3);
                              }

Script Include


                              <script src="./rocketbelt.progress-indicators.min.js"></script>

Examples


                                    <button class="button button-primary is-busy is-busy-inverted"><span>Click to Trigger Busy</span></button>

                                    <div class="form-group is-busyable">
                                      <textarea id="busy-textarea" placeholder="Blur the textarea to trigger a busy state."></textarea>
                                      <label for="busy-textarea">Message</label>
                                    </div>
                                    <ul class="radio-group radio-group-chunky is-busyable">
                                      <li>
                                        <input id="busy-chunky-1" type="radio" name="radio-group-chunky">
                                        <label for="busy-chunky-1">Click an option to trigger a busy state.</label>
                                      </li>
                                      <li>
                                        <input id="busy-chunky-2" type="radio" name="radio-group-chunky">
                                        <label for="busy-chunky-2">Click an option to trigger a busy state.</label>
                                      </li>
                                    </ul>
                                    <div class="form-group is-busyable">
                                      <select id="busy-select">
                                        <option value="" disabled="" selected="" hidden="">Change selection to trigger busy</option>
                                        <option>Biblioteca</option>
                                        <option>Marcel</option>
                                        <option>Pierce</option>
                                      </select>
                                      <label for="busy-select">Collection</label>
                                    </div>

Page-Level


                              <div class="form-group is-busyable is-busyable-page" style="width: 100%;">
                                <textarea class="is-busy_trigger" id="busy-large" placeholder="Blur the textarea to trigger a busy state." style="height: 50rem;"></textarea>
                                <label for="busy-large">Message</label>
                              </div>
